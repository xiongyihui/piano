Fruit Piano

This piano is built with a basket of fruit, a microcontroller and a computer.

The idea is from Makey Makey, but uses different hardware and software. An mbed enabled microcontroller with USB is used. The software running in the microcontroller is written by C++ using mbed C/C++ SDK. This site is forked from https://github.com/mrcoles/javascript-piano.
